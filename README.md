# Install

with yarn:
```
$ yarn global add @oclif/example-plugin-ts
```

or with npm:
```
$ npm install -g @oclif/example-plugin-ts
```

# Usage

```sh-session
$ oclif-example COMMAND
running command...
$ oclif-example (-v|--version|version)
@oclif/example-plugin-ts/1.3.5 (linux-x64) node-v9.5.0
$ oclif-example --help [COMMAND]
USAGE
  $ oclif-example COMMAND
...
```

# Commands

* [oclif-example hello [FILE]](#hello-file)
## hello [FILE]

describe the command here

```
USAGE
  $ oclif-example hello [FILE]

OPTIONS
  -f, --force
  -n, --name=name  name to print

EXAMPLES
  $ oclif-example hello
  hello world from ./src/hello.ts!

  $ oclif-example hello --name myname
  hello myname from .src/hello.ts!
```

_See code: [src/commands/hello.ts](https://github.com/oclif/example-plugin-ts/blob/v1.3.5/src/commands/hello.ts)_
